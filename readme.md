## Problem 2: Fancy Form

Problem 2 solution is available as a separate repository: [fancy-form](https://github.com/Danhyeye/fancy-form.git)

This is a React + TypeScript + Vite project that implements a fancy swap form interface.

### How to Run the Project

1. **Clone the repository:**
   ```bash
   git clone https://github.com/Danhyeye/fancy-form.git
   cd fancy-form
   ```

2. **Install dependencies:**
   ```bash
   npm install
   ```

3. **Start the development server:**
   ```bash
   npm run dev
   ```

4. **Build for production:**
   ```bash
   npm run build
   ```

5. **Preview production build:**
   ```bash
   npm run preview
   ```

The development server will start and you can access the application at `http://localhost:5173` (or the port shown in the terminal).
